Accelerometer evaluation board

Description:

Accelerometer as system sensor input e.g. user input and control, movement detection, slipping control, etc. PCB was developed based on reference schematic provided by STMicroelectronics for theirs digital MEMS accelerometer LIS331DL.

Alarm clock with light effects for deaf people

Description:

Main goal of this project was to introduce different solution for alarm clock output in contrary to currently available products which produce extremely loud noise as output with no concern for other people in the same living environment, e.g. spouse, children, family members etc.

Remote RF control, 2,4GHz ISM band

Description:

Device provides RF control of remote devices taking input from widely spread PlayStation joysticks. Further control can be accomplished using PC software and USB connectivity supported by STMicroelectronics STM32F devices. Device enables custom channel implementation and telemetry data transfer.

Handheld PID regulator

Description:

Real time graphing on color LCD of measured temperature. Touch screen enables simple user alternations of heating profile. Heating profiles and measured data can be stored on uSD card. Device is powered from Li-Po battery. Microcontroller is driving Solid-State-Relay for controlling load.
